## If Something Goes Wrong

1. **Check ports**: `lsof -i :3000` and `lsof -i :5000`
2. **Kill stuck processes**: `npm run kill-ports`
3. **Run diagnostics**: `npm run diagnose`
4. **Fix dependencies**: `npm run fix`
5. **Last resort**: `npm run reset`

## System Check

```bash
# Check Node.js

# Check npm

# Check ports

df -h .              # Need 500MB+ free
```

## API Endpoints

```
POST   /api/quiz/start       Start game
GET    /api/quiz/question    Get question
POST   /api/quiz/answer      Submit answer
GET    /api/quiz/stats       Get stats
POST   /api/quiz/reset       Reset game
GET    /api/health           Health check
```
